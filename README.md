# sakoloda

sakolodaは、画像アップローダーです。

## なぜ作ったか

imgurが迷惑がってるから。

## 特徴

- 複数ファイルのドラッグアンドドロップ対応。
- クリップボードからの貼り付け対応。

## 展望

- まだ削除機能がない（いる？）
